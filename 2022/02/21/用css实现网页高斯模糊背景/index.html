用css实现网页高斯模糊特效


Attention: 这篇文章探讨的是如何实现背景高斯模糊，而不是整个视图高斯模糊！ ！ ！


大概效果是这样：


没有高斯模糊


使用高斯模糊效果



怎么样？ ？是不是感觉好漂亮的效果，那么如何实现呢(¯︶¯)


先来看看DOM结构

其实有两种方法，这是方法一，相容性较好！



-> 外层有一个 id=animeDetails 的Div标签，用于撑起整个页面的高度


-> 内层使用了 class=animeDetails_mask 的一个Div标签用于放入要展示的内容，例如现在我放入的是contents这几个字


-> 现在开始设定背景图及高斯模糊，用到了伪元素唷（≧∇≦）




!!!终于，完成惹，来看看效果吧



好像，真的有高斯模糊了，但是为什么图会不透明度这么低，让我怎么看字啊！

对惹！在 Div.animeDetails_mask 上设定背景色就好了吧！

不管用欸，这是怎么回事呢？

其实答案很简单， ::before 使用了position: absolute，已经在文档流最上方了，会吧后面的背景色遮住。

解决方案就是，用 ::after 设定背景色



-> 成果预览

-> 给 #animeDetails 设定背景图

-> 给 .animeDetails_mask 设定样式
